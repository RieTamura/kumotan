# App Store プライバシー宣言ガイド

**作成日**: 2026-02-19
**対象バージョン**: v1.0.0

App Store Connect の「App Privacy」セクションへの入力内容をまとめたドキュメント。

---

## 1. トラッキングの申告

**申告内容**: 「このアプリはユーザーをトラッキングしません」

アプリは広告目的・サードパーティ提供目的のトラッキングを一切行わない。

---

## 2. データ収集の申告

App Store Connect では「収集するデータの種類」を選択し、各データに対して以下を申告する：
- 使用目的
- ユーザーのIDと紐づくか
- トラッキング目的か

---

### 2-1. 収集するデータ

#### ✅ 識別子（Identifiers）
| データ | 具体的な内容 |
|--------|-------------|
| ユーザーID | Bluesky DID（`did:plc:...`）、Bluesky ハンドル |

- **使用目的**: 認証（App Functionality）
- **ユーザーIDと紐づく**: Yes
- **トラッキング**: No
- **補足**: Bluesky OAuth 認証で取得。デバイスの SecureStore（iOS Keychain）に保存。

---

#### ✅ ユーザーコンテンツ（User Content）
| データ | 具体的な内容 |
|--------|-------------|
| その他のユーザーコンテンツ | 保存した英単語・日本語訳・定義・投稿URL・投稿テキスト |

- **使用目的**: App Functionality（単語帳のクロスデバイス同期・バックアップ）
- **ユーザーIDと紐づく**: Yes
- **トラッキング**: No
- **補足**: ユーザーが明示的に操作した場合のみ、ユーザー自身の Bluesky PDS（Personal Data Server）に保存される。アプリ開発者のサーバーへは送信されない。

---

#### ✅ 検索履歴（Search History）
| データ | 具体的な内容 |
|--------|-------------|
| 検索履歴 | 辞書で検索した英単語 |

- **使用目的**: App Functionality（辞書検索・翻訳）
- **ユーザーIDと紐づく**: No
- **トラッキング**: No
- **補足**:
  - Free Dictionary API（`api.dictionaryapi.dev`）へ匿名でリクエスト送信
  - DeepL API・Yahoo! JAPAN API へもリクエスト送信されるが、**ユーザー自身が入力した API キーを使用**するため、アプリ開発者はこれらの通信内容にアクセスできない

---

#### ✅ フィードバックデータ（Other Data）
| データ | 具体的な内容 |
|--------|-------------|
| その他のデータ | ユーザーが入力したフィードバックテキスト（不具合報告・機能要望・辞書修正） |

- **使用目的**: App Functionality（辞書品質改善・バグ修正）
- **ユーザーIDと紐づく**: No
- **トラッキング**: No
- **補足**: Google Apps Script 経由で送信。ユーザーの Bluesky 認証情報は含まれない。フィードバック内に単語の投稿 URL が含まれる場合がある。

---

### 2-2. 収集しないデータ（申告不要）

以下のデータは**一切収集しない**：

| カテゴリ | 理由 |
|---------|------|
| 連絡先情報（メール・電話・住所） | 収集なし |
| 健康・フィットネス情報 | 収集なし |
| 財務情報 | 収集なし |
| 位置情報（精確・おおよそ） | 収集なし |
| 連絡先リスト | 収集なし |
| 閲覧履歴 | 収集なし（Bluesky タイムラインはオンメモリのみ、DB に永続化しない） |
| 診断情報（クラッシュレポート） | Sentry・Firebase 等の外部クラッシュレポートサービス不使用 |
| 広告データ | 収集なし |

## 3. 外部送信先一覧

| 外部サービス | 送信されるデータ | 認証情報 | 目的 |
|-------------|----------------|---------|------|
| Bluesky / bsky.social | Bluesky ハンドル、OAuth トークン | あり（ユーザーのOAuth認証） | 認証・タイムライン取得 |
| PLC Directory（plc.directory） | ユーザー DID | なし（公開情報） | DID解決 |
| ユーザー自身の PDS | 単語帳データ | あり（ユーザーのOAuthトークン） | 単語帳バックアップ・同期 |
| Free Dictionary API | 英単語 | なし（匿名） | 英語定義取得 |
| DeepL API | 英語テキスト | ユーザー自身の API キー | 日本語翻訳 |
| Yahoo! JAPAN Text Analysis API | 日本語テキスト | ユーザー自身の Client ID | 形態素解析・ふりがな |
| GitHub Pages（rietamura.github.io） | なし（GETリクエストのみ） | なし | JMdict辞書データ配信 |
| Google Apps Script | フィードバックテキスト | なし | フィードバック収集 |

---

## 4. App Store Connect 入力手順

1. App Store Connect > 対象アプリ > 「App Privacy」
2. 「プライバシー慣行を管理する」をクリック
3. **「このアプリはユーザーをトラッキングしません」** を選択
4. 収集するデータの種類で以下を選択：
   - ✅ 識別子
   - ✅ ユーザーコンテンツ
   - ✅ 検索履歴
   - ✅ その他のデータ
5. 各カテゴリで上記の申告内容を入力
