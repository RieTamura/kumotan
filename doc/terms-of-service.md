# 利用規約

本利用規約（以下「本規約」といいます。）は、くもたん（以下「本アプリ」といいます。）の利用条件を定めるものです。本アプリを利用する全てのユーザー（以下「ユーザー」といいます。）は、本規約に同意の上、本規約に従って本アプリを利用するものとします。

## 第1条（適用）
1. 本規約は、ユーザーと本アプリの運営者（以下「運営者」といいます。）との間の本アプリの利用に関わる一切の関係に適用されるものとします。
2. 運営者は本アプリに関し、本規約のほか、ご利用にあたってのルール等、各種の規定（以下「個別規定」といいます。）を定めることがあります。これら個別規定はその名称のいかんに関わらず、本規約の一部を構成するものとします。

## 第2条（サービスの定義）
1. 本アプリは、AT Protocol（Bluesky等）のタイムラインを閲覧しながら、英語の単語・熟語を日本語で学習するための語彙学習支援ツールです。
2. 本アプリは、以下の主な機能を提供します。
   - Blueskyタイムラインの表示
   - 投稿内の英単語の辞書検索・翻訳
   - 単語帳への登録・管理
   - 学習進捗の記録・表示
   - 学習進捗のBlueskyへのシェア
3. 本アプリの翻訳機能およびテキスト解析機能は、ユーザー自身が取得した外部サービスのAPIキー（DeepL API Key、Yahoo! JAPAN Client ID）を利用します。これらのAPIキーの取得・管理はユーザーの責任において行うものとします。

## 第3条（利用登録）
1. 本アプリの利用には、Blueskyアカウントによるログインが必要です。ユーザーは、AT Protocol OAuth認証を通じてBlueskyアカウントでログインすることにより、本アプリの利用を開始するものとします。
2. 運営者は、本アプリの利用に際して、ユーザーのBlueskyアカウントのパスワード等の認証情報を取得・保存することはありません。

## 第4条（アカウントの管理）
1. ユーザーは、自己の責任において、本アプリの利用に使用するBlueskyアカウントを適切に管理するものとします。
2. ユーザーのBlueskyアカウントが第三者によって不正に使用されたことによって生じた損害について、運営者は一切の責任を負わないものとします。
3. ユーザーが本アプリに設定した外部サービスのAPIキー（DeepL API Key、Yahoo! JAPAN Client ID）の管理は、ユーザーの責任において行うものとします。APIキーの漏洩・不正使用等によって生じた損害について、運営者は一切の責任を負わないものとします。

## 第5条（データの保存と管理）
1. 本アプリにおけるユーザーデータ（登録した単語、学習統計等）は、ユーザーの端末内にローカルに保存されます。運営者のサーバーにユーザーデータが保存されることはありません。
2. 本アプリは、AT Protocolの仕組みに基づき、登録した単語データおよび学習進捗をユーザー自身のPDS（Personal Data Server）に同期する機能を提供します。PDSに保存されたデータは、ユーザー自身が所有・管理するものであり、運営者がこれを管理・閲覧することはありません。
3. PDSへのデータ同期は、端末変更やアプリ再インストール時のデータ復元を目的としています。PDS側の障害やデータ消失について、運営者は一切の責任を負わないものとします。
4. 端末の故障、アプリの削除、その他の事由によりローカルデータが失われた場合、運営者はデータの復旧について一切の責任を負わないものとします。
5. ユーザーは、必要に応じて本アプリのデータエクスポート機能を利用し、自己の責任においてデータのバックアップを行うものとします。

## 第8条（外部サービスの利用）
1. 本アプリは、以下の外部サービスおよびデータを利用して機能を提供します。
   - Bluesky / AT Protocol（タイムライン取得、投稿機能）
   - DeepL API（翻訳機能、ユーザーが自身のAPIキーを設定した場合）
   - Yahoo! JAPAN テキスト解析API（日本語解析機能、ユーザーが自身のClient IDを設定した場合）
   - Free Dictionary API（英語辞書機能）
